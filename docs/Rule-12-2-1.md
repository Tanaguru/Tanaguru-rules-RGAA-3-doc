 *  Le <a href="http://references.modernisation.gouv.fr/sites/default/files/RGAA3_RC2-1/glossaire.htm#mMenuNav">menu de navigation</a> est toujours &agrave; la m&ecirc;me place dans la pr&eacute;sentation
